# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`

- `GET /api/users/count`

- `GET /api/users/check`

- `GET /api/users/search`

### Session

- `POST /api/session`

- `DELETE /api/session`

- `GET /api/session`


### Channels

- `GET /api/channels`

- `GET /api/channels/count`

- `POST /channels/disconnect/`

- `POST /channels/connect/`

- `POST /api/channels`

- `PATCH /api/channels/:id`


### Messages

- `GET /api/channels/:id/messages`

- `POST /api/messages`

- `DELETE /api/messages/:id`
